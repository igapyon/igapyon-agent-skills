# Project Plan

## 目標

- 個人用 Agent Skills リポジトリを整理された状態で維持する
- 明確さを失わずに skill の対象範囲を広げる
- 人間にも LLM にもたどりやすい skill 文書を保つ

## マイルストーン

- リポジトリ構成を整備済み
- 同期ワークフローを整備済み
- 技術系と音楽系の writing coach skill を references 付きで整理済み
- `docs/` 配下に LLM 向け作業文書を導入済み

## 今後の作業

- 必要に応じて skill を追加する
- `.codex/skills/` への同期が本当に必要か見極めてから実装する
- UI metadata が必要になったら `agents/openai.yaml` を検討する
- 実運用に合わせて style guide を継続的に調整する
